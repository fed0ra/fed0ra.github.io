    Hexo landscape主题优化

    1.在hexo中加入多说评论

    首先在主题根目录下_config.yml中增加duoshuo_shortname: 你的多说二级域名
    如果使用的是默认的landscape主题只需要修改themes\landscape\layout\_partial\article.ejs中的

    <% if (!index && post.comments && config.disqus_shortname){ %>
 <section id="comments">
   <div id="disqus_thread">
     <noscript>Please enable JavaScript to view the <a href="//disqus.com/?ref_noscript">comments powered by Disqus.</a></noscript>
   </div>
 </section>
 <% } %>

    改为以下代码：
    　　修改时候注意项：
    　　　　1.注册多说后，从后台管理找到创建的站点-->工具-->获取代码-->通用代码，拷贝代码，将拷贝下的代码粘贴到并替换上面代码中<div></div>标签内容。
    　　　　2.修改源代码<% %>中获取后台的变量值由config.disqus_shortname，改为config.duoshuo_shortname，即之前配置文件中增加的变量名。
    　　　　3.拷贝过来的代码中的short_name也可改成变量<%= theme.duoshuo_shortname %>

    <% if (!index && post.comments && (config.duoshuo_shortname || theme.duoshuo_shortname)){ %>
<section id="comments">
    <!-- 多说评论框 start -->
    <div id="ds-thread" class="ds-thread" data-thread-key="<%= post.path %>" data-title="<%= post.title %>" data-url="<%= post.permalink %>"></div>
    <!-- 多说评论框 end -->
    <!-- 多说公共JS代码 start (一个网页只需插入一次) -->
    <script type="text/javascript">
    var duoshuoQuery = {short_name:"<%= theme.duoshuo_shortname %>"};
        (function() {
            var ds = document.createElement('script');
            ds.type = 'text/javascript';ds.async = true;
            ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
            ds.charset = 'UTF-8';
            (document.getElementsByTagName('head')[0] 
             || document.getElementsByTagName('body')[0]).appendChild(ds);
        })();
        </script>
    <!-- 多说公共JS代码 end -->
</section>
<% } %>

    如果是其他主题，也只需要修改主题\layout\_partial\comment.ejs
    注意：使用多说的话，Thread Key一定不要改变，Thread Key相当于是识别码；如果你更新了Hexo，文章的Thread Key改变了的话，那么评论就会没有。

    2.在hexo中添加多说最新评论边栏

    前提安装多说评论

    • 在themes\landscape\layout\_widget目录下新建duoshuo_recent_comments.ejs，添加以下内容：

      <% if (site.posts.length){ %>
  <div class="widget-wrap">
    <h3 class="widget-title"><%= __('最新评论') %></h3>
    <div class="widget">
      <ul>
<!-- 多说最新评论 start -->
    <div class="ds-recent-comments" data-num-items="5" data-show-avatars="1" data-show-time="1" data-show-title="1" data-show-admin="1" data-excerpt-length="70"></div>
<!-- 多说最新评论 end -->
<!-- 多说公共JS代码 start (一个网页只需插入一次) -->
<script type="text/javascript">
var duoshuoQuery = {short_name:"<%= theme.duoshuo_shortname %>"};
    (function() {
        var ds = document.createElement('script');
        ds.type = 'text/javascript';ds.async = true;
        ds.src = (document.location.protocol == 'https:' ? 'https:' : 'http:') + '//static.duoshuo.com/embed.js';
        ds.charset = 'UTF-8';
        (document.getElementsByTagName('head')[0] 
         || document.getElementsByTagName('body')[0]).appendChild(ds);
    })();
    </script>
<!-- 多说公共JS代码 end -->
      </ul>
    </div>
  </div>
<% } %>

    • 在主题目录中的_config.yml下添加duoshuo_recent_comments：

      widgets:
- duoshuo_recent_comments

    3.添加访问次数统计功能

    使用不蒜子，修改themes\landscape\layout\_partial\footer.ejs，添加上述脚本即可，也可以添加到header中。

    <script async src="//dn-lbstatics.qbox.me/busuanzi/2.3/busuanzi.pure.mini.js"></script>
<span id="busuanzi_container_site_pv">本站总访问量<span id="busuanzi_value_site_pv"></span>次</span>

    版权声明：

    参考不如的博文：http://ibruce.info/2015/04/04/busuanzi/

    4.主页文章显示摘要

    编辑md文件的时候，在要作为摘要的文字后面添加<!--more-->\即可。

    5.加入「fork me on github」

    github给出的GitHub Ribbons教程，把代码插入到任意一个全局的模板文件中就行，比如themes\landscape\layout\_partial\header.ejs的末尾。

    6.添加分类、标签云、友情链接widget

    添加分类、标签云widget

    在themes\landscape\_config.yml中，默认开启，如没开启请修改如下：

    widgets:
- category
- tagcloud

    添加友情链接widget

    在themes\landscape\layout\_widget中新建名为links.ejs的文件，编辑内容如下：

    <% if (site.posts.length){ %>
  <div class="widget-wrap">
    <h3 class="widget-title"><%= __('links') %></h3>
    <div class="widget">
      <ul>
        <% for (var i in (config.links || theme.links)){ %>
          <li>
            <a href="<%- theme.links[i] %>" target="_blank"><%= i %></a>
          </li>
        <% } %>
      </ul>
    </div>
  </div>
<% } %>

    在themes\landscape\_config.yml中，添加如下：

    # Links
links:
    explainshell: http://explainshell.com    #随意修改
    linux命令手册: http://linux.51yip.com
    tool.oschina: http://tool.oschina.net
    Bootcss: http://www.bootcss.com
    google替代品: http://www.gycc.com
    Youtube downloader: http://www.clipconverter.cc

    7.添加新浪微博widget(微博秀)

    1. 去新浪微博秀设置和生成微博秀代码。

    2. 在themes\landscape\layout\_widget中新建名为weibo.ejs的文件，将刚才的代码直接保存到这里。

    3. 在themes\landscape\_config.yml中，添加如下：

      widgets:    #站点右边栏，暂时默认
- category
- tag
- tagcloud    #标签云
- archive
- recent_posts    #最新帖子
- weibo        #新浪微博秀
