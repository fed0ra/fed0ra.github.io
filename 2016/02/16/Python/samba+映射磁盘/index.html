    samba+映射磁盘


    Welcome to Fedor’s blog! If you find any problems, you can tell me in any way.

    Linux运维开发人员要做自动化，很多时候代码是不能在IDE上执行的，而是要放到Linux系统下运行，而每次运行代码，都需要先登陆服务器，再将代码上传到虚拟机上，然后运行的做法，麻烦而又效率低。

    在windows上开发，代码Linux下运行的方法：

    1. 需要一个安装好的Ubuntu系统，初始化完毕虚拟机，并安装好samba服务

    samba的安装配置使用搜索引擎可以办到，命令如下：

    $ sudo su -
$ sudo apt-get insall samba
$ sudo apt-get install smbfs
$ sudo mkdir -p /home/share/
$ sudo chmod 777 /home/share/
$ suco cp /etc/samba/smb.conf /etc/samba/smb.conf.bak
$ sudo vim /etc/samba/smb.conf

    编辑配置文件/etc/samba/smb.conf，添加如下内容：

    [share]
  comment = Ubuntu share
  browseable = yes
  public = yes
  path =  /home/share
  writable = yes
  guest ok = yes

    2. 登录samba目录


    3. 登录进去后，右键点击共享目录(share)，点击映射网络驱动器映射个盘符Z

    4. 现在可以在我的电脑中找到Z盘符，将代码拖动到Z盘下即会同步到虚拟机的相应目录(/home/share)下，或者将IDE的工作目录直接设置在Z盘符下。IDE推荐使用PyCharm。

    附：使用eclipse将工作目录设置在映射盘符方法：

    打开eclipse，会有选择workspace的界面，选中映射的磁盘


    如果没有这个界面，就是之前勾选了Use this the default and do not ask again，在不关闭eclipse前提下，打开第二个eclipse，它会重新让你选择workspace，再选择workspace为映射的盘符，注意这次不要勾选Use this the default and do not ask again了。

    最后，在打开eclipse的中，映射python解释器：

    • 在Window菜单中选Preference
    • PyDev->Interpreter - Python，New一个Python解释器，填上解释器名字和路径，路径选相应的python.exe。
