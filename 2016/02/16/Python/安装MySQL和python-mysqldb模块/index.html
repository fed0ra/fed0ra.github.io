安装MySQL和python-mysqldb模块


Welcome to Fedor’s blog! If you find any problems, you can tell me in any way.

1. Linux环境

有时候开发是在windows上，而运行代码则需要在Linux系统上，所以需要Linux环境也需要安装MySQL数据库和python-mysqldb模块

安装MySQL数据库和python-mysql模块，命令如下：

$ apt-get install mysql-server python-mysqldb    \\提示输入mysql密码

2. Windows环境

2.1.1 使用安装包安装MySQL

  • MySQL官网下载安装包mysql_installer_community_V5.6.21.1_setup.1415604646.msi，一步步安装

2.1.2 使用MySQL免安装版安装

  • MySQL官网Download-->Community-->MySQL Community Server，下载 mysql-5.5.47-winx64.zip安装包
  • 解压，将以下载的MySQL压缩包解压到自定义目录下,我的解压目录是："F:\MySQL\mysql-5.5.47-winx64"
  • 将解压目录下默认配置文件 my-default.ini拷贝一份，改名my.ini（如果没有my-default.ini,可自己新建my.ini或者从其他地方中获取）

  • 添加环境变量，操作如下：

    a)    右键单击我的电脑->属性->高级系统设置(高级)->环境变量
    点击系统变量下的新建按钮
    输入变量名：MYSQL_HOME
    输入变量值：F:\MySQL\mysql-5.5.47-winx64
    #即为mysql的自定义解压目录。
b)    选择系统变量中的Path
    点击编辑按钮
    在变量值中添加变量值：%MYSQL_HOME%\bin
    注意是在原有变量值后面加上这个变量，用;隔开，不能删除原来的变量值，

  • 将mysql注册为windows系统服务

    a)从控制台进入（管理员身份运行）到MySQL解压目录下的 bin 目录下：
b)输入服务安装命令：
    mysqld.exe install MySQL --defaults-file="F:\MySQL\mysql-5.5.47-winx64\my.ini"    \\解压目录下修改的my.ini文件

安装成功后会提示服务安装成功。移除服务命令为：mysqld remove

注1：my.ini文件放在MySQL解压后的根目录下
注2：在windos的cmd下安装mysql，切换到mysql的bin目录下面执行：mysqld --install
报错信息如下：Install/Remove of the Service Denied
解决办法：打开cmd.exe程序的时候选择“用管理员身份打开”。

  • 启动MySQL服务

    方法一：
    启动服务命令为：net start mysql
方法二：
    打开管理工具 服务，找到MySQL服务。
    通过右键选择启动或者直接点击左边的启动来启动服务。

  • 修改root账号的密码（刚安装完成时root账号默认密码为空，此时可以将密码修改为指定的密码。如：123456）

    c:>mysql –uroot
mysql>show databases;
mysql>use mysql;
mysql>UPDATE user SET password=PASSWORD("123456") WHERE user='root';
mysql>FLUSH PRIVILEGES;
mysql>QUIT

2.2 安装python-mysql模块：MySQL-python-1.2.5.win32-py2.7

  • 前往Python官网–>PyPI–>search MySQL-python，下载MySQL-python-1.2.5.win32-py2.7.exe，下一步下一步安装。

  • 安装MySQL-python-1.2.5.win32-py2.7报错

    报错：Python version 2.7 required,which was not found in the registry.
    解决办法：下载registry.py脚本，把脚本放到任意一盘中执行，运行–>cmd，执行python registry.py
    例：registry.py脚本在F:\MySQL\data目录下

    >f:    \\F盘
>cd MySQL/data \\切换到脚本所在路径
>python registry.py \\执行脚本
python 2.7 is already registered    \\OK

2.3安装WindowsMySQL数据库管理工具：Navicat_for_MySQL

  • Navicat数据库管理工具，搜索引擎搜索Navicat，下载安装，过程简单，略。

2.4 测试模块安装成功与否

导入MySQLdb测试是否安装成功(Windows/Linux同),如果没报错证明安装成功。

C:\Users\my-pc>python
Python 2.7.10 (default, May 23 2015, 09:44:00) [MSC v.1500 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> import MySQLdb
>>>
