    安装Django、django-rest-framework和django-debugtools


    Welcome to Fedor’s blog! If you find any problems, you can tell me in any way.

    Django是一个开放源代码的Web应用框架，由Python写成。采用了MVC的框架模式，即模型M，视图V和控制器C。它最初是被开发来用于管理劳伦斯出版集团旗下的一些以新闻内容为主的网站的，即是CMS（内容管理系统）软件。

    django-debugtools调试工具模块，它提供了一些易于使用的调试程序来协助Django开发。

    Django REST Framework，就是基于Django的REST框架，可以轻松部署web APIs。

    1. pip方法安装Django和django-rest-framework

    pip install django
pip install djangorestframework
pip install django-debugtools

    2. 源码安装Django

    Django下载后为压缩包，解压缩后，在dos下进入到Django-1.8.14（读者可能版本不同）目录，执行python setup.py install，然后开始安装，Django将要被安装到Python的Lib下site-packages。

    检查是否安装成功，进入dos输入：

    • 1.输入python
    • 2.输入import django //不报错能导入即安装成功
    • 3.输入django.get_version() //查看django版本

    3. 源码安装django-debugtools

    项目githup地址：https://github.com/edoburu/django-debugtools

    下载django-debugtools，解压缩后，进入到解压目录，执行python setup.py install，然后开始安装，django-debugtools将要被安装到Python的Lib下site-packages。

    4. 源码安装django-rest-framework

    在github（地址在下文）上下载django-rest-framework-master.zip的压缩包，解压缩后，在dos下进入到django-rest-framework-master目录，执行python setup.py install，然后开始安装，django-rest-framework将要被安装到Python的Lib下site-packages。

    django-rest-framework官网：http://www.django-rest-framework.org/
    项目githup地址：https://github.com/tomchristie/django-rest-framework

    RESTful规范相关参考：
    理解RESTful架构：http://www.ruanyifeng.com/blog/2011/09/restful
    RESTful API设计指南：http://www.ruanyifeng.com/blog/2014/05/restful_api.html
