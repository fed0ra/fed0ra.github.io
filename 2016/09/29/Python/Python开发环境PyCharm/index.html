    Python开发环境PyCharm


    Welcome to Fedor’s blog! If you find any problems, you can tell me in any way.

    一、准备工作：

    IDE选择: PyCharm

    1. 下载Python：

    • 下载地址：http://www.python.org/
    • Python有Python2和Python3两个版本，语法有些区别。 可以安装双版本（http://edu.51cto.com/course/course_id-6823.html）。

    2. 下载PyCharm：

    • 下载地址：https://www.jetbrains.com/pycharm/。

    3. 安装Python和PyCharm

    • PyCharm和Python的安装很简单，直接下一步。

    返回顶部

    二、PyCharm基本配置

    注：以下所有设置均在File>Settings下操作

    1.在Pycharm下为你的Python项目配置Python解释器

    Project:当前项目名>Project Interpreter>add Local选择python安装目录下`python.exe`

    2.Pycharm基本设置，如字体颜色、主题、脚本头设置、显示行号等。

    1. 不使用tab，将tab设置成4个空格：Editor>Code Style>Python
2. 字体、字体颜色：Edit>Colors & Fonts>Python
3. 关闭自动更新：Appearance & Behavior>System Settings>Updates取消勾选
4. 脚本头设置：Edit>File and Code Templates>Python Script 
5. 显示行号：Edit>General>Appearance>Show line numbers勾选
6. 右侧竖线是PEP8的代码规范，提示一行不要超过120个字符，默认存在

    3.编码UTF-8

    1. Editor>File Encodings>
将IDE Encoding/Project Encoding/Default encoding for properties file均设成UTF-8
2. 脚本头添加`# _*_coding:utf-8_*_`，建议在第4条脚本头设置中写好模板

    4.SSH Terminal：Default encoding:UTF-8

    Tools>SSH Terminal>最后一行Default encoding:选择UTF-8

    5.Git配置

    1. 需要本地安装好Git
2. Version Control>Git>Path to Git executable选择Git安装路径下`git.exe`
3. 配置了Git等版本控制系统之后，可以很方便的diff查看文件的不用

    6.使用Pycharm安装Python第三方模块

    Project:当前项目名>Project Interpreter>点击右侧绿色小加号搜索安装

    6.Pycharm安装插件，例如Markdown support、数据库支持插件等。

    1. Plugins>Browse repositories>Browse Repositories
2. 搜索插件‘markdown support’>install

    返回顶部

    三、导出、导入自定义的配置

    1. 导出自定义配置：
File>Export Settings
2. 导入自定义配置：
File>Import Settings

    返回顶部

    四、PyCharm基本使用

    1.在Pycharm下创建Python文件、Python模块

    1. File>New>Python File
2. File>New>Python Package

    2.常用操作，如复制文件路径、在文件管理器中打开等。

    1. 复制文件路径：左侧文件列表右键选中的文件>Copy Path
2. 在文件管理器中打开：右键选中的文件>往下找到Show In Explorer
3. 快速定位：Ctrl+左键点击内建模块，会在源文件中展开
4. 查看结构：IDE左侧边栏Structure查看当前项目的结构
5. tab批量换space：Edit>Convert Indents
6. TODO的使用：# TODO 要记录的事情
7. Debug设置断点，直接点击行号与代码之间的空白处即可设置断点
8. Tab页上右键>Move Right（Down），把当前Tab页移到窗口右边（下边），方便对比
9. 文件中右键>Local History能够查看文件修改前后的对比
10. IDE右下角能看到一些有用的信息
如：光标当前在第几行的第几个字符、当前回车换行、当前编码类型、当前Git分支
11. IDE右侧边栏>Database

    3.常用快捷键

    0. 常用快捷键的查询和配置：Keymap
1. Ctrl + D：复制当前行
2. Ctrl + E：删除当前行
3. Shift + Enter：快速换行
4. Ctrl + /：快速注释（选中多行后可以批量注释）
5. Tab：缩进当前行（选中多行后可以批量缩进）
6. Shift + Tab：取消缩进（选中多行后可以批量取消缩进）
7. Ctrl + F：查找
8. Ctrl + R：替换

    4.去掉代码中的波浪线，需要遵守PEP8编码规范

    1. 单独一行的注释：#+1空格+注释内容
2. 代码后跟着的注释：2空格+#+1空格+注释内容
3. ...等
