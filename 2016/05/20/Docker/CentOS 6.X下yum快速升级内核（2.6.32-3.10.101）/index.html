CentOS 6.X下yum快速升级内核（2.6.32-3.10.101）


Welcome to Fedor’s blog! If you find any problems, you can tell me in any way.

一、准备工作

3条命令搞定内核升级，亲测CentOS 6.7下没问题。

rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org
rpm -Uvh http://www.elrepo.org/elrepo-release-6-6.el6.elrepo.noarch.rpm
yum --enablerepo=elrepo-kernel install kernel-lt -y && reboot

Step1: 确认内核版本

[root@CentOS-Server02 ~]# uname -r
2.6.32-573.18.1.el6.x86_64
[root@CentOS-Server02 ~]# cat /etc/centos-release 
CentOS release 6.7 (Final)

Step 2: 导入public key

[root@CentOS-Server02 ~]# rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org

导入失败，但执行后面命令依然升级成功。

Step 3: 安装ELRepo

[root@CentOS-Server02 ~]# rpm -Uvh http://www.elrepo.org/elrepo-release-6-6.el6.elrepo.noarch.rpm

Step 4: 安装长期支持版本

[root@CentOS-Server02 ~]# yum --enablerepo=elrepo-kernel install kernel-lt -y

Step 5: 编辑grub.conf文件，修改Grub引导顺序

新装的内核位置为0，所以将default修改为0，保存退出重启。
[root@CentOS-Server02 ~]# vi /etc/grub.conf
default=0

[root@CentOS-Server02 ~]# reboot

Step 6: 检查内核版本

[root@CentOS-Server02 ~]# uname -r
3.10.101-1.el6.elrepo.x86_64
