清除Elasticsearch旧记录


Welcome to Fedor’s blog! If you find any problems, you can tell me in any way.

1.使用curator清除Elasticsearch旧记录

curator是一个开源的项目，它是一个命令，可以管理Elasticsearch中的索引，包括删除、关闭、打开它们。
github地址：https://github.com/elastic/curator
官方文档：https://www.elastic.co/guide/en/elasticsearch/client/curator/current/index.html

安装curator

# yum install python-pip -y
# pip install elasticsearch-curator
# which curator
/usr/bin/curator

指定版本安装

# pip install -U --user elasticsearch-curator==3.5.1

卸载

# pip uninstall  elasticsearch-curator

使用curator，可参考官方文档，或查看帮助/usr/bin/curator --help

# /usr/bin/curator --host 182.92.157.42 --port 9200 delete --disk-space 1 indices --all-indices

--disk-space 1是删除大于1GB的索引，单位GB
--all-indices所有索引
